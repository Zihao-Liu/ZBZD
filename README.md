# ZBZD
---
## 简介
使用SpringMVC，Spring，MyBatis，JSP实现一个类似于知乎的网站。

## 功能列表：
---
* [x] V0.0：实现了用户注册/登录功能。 
* [x] V0.1：实现了发布问题功能。
   - [x] V0.1.1：对未登录用户的提问动作进行拦截。
   - [x] V0.1.2：用户可以在发布问题时选择匿名发布。
   - [x] V0.1.3：用户发布问题时可以选择话题。
   - [ ] V0.1.4：用户可以在话题中查看问题。
   - [ ] V0.1.5：用户可以改变自己问题的匿名状态。
   - [ ] V0.1.6：用户可以查看自己发布的问题。   


## 需要改进：
